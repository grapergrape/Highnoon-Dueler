# High Noon Card Structure

## Card Types

- Feats are immediate tricks, tactics, reactions, and bursts of pressure.
- Guns are equipped weapons. Each class starter gun sits outside the deck as the active gun.
- Stances stay in play during a duel and provide ongoing modifiers.
- Showdowns are one-per-deck build-around cards that start at Level I and advance after each unresolved High Noon until Level III.

## Baseline Deck

Each starter deck contains 24 cards:

| Type | Count |
| --- | ---: |
| Feats | 16 |
| Stances | 5 |
| Extra Guns | 2 |
| Showdowns | 1 |
| Total | 24 |

The class starter gun is not counted in the 24 cards.

| Type | Count |
| --- | ---: |
| Starter Gun | 1 |

| Feats | 14 |
| Stances | 5 |
| Showdowns | 2 |
| Total | 25 |

Recommended shared pool target:

| Type | Count |
| --- | ---: |
| Feats | 24 |
| Guns | 6 |
| Stances | 8 |
| Showdowns | 2 |
| Total | 40 |

Decks should generally run 16-18 class cards and 6-8 shared cards, with 8 shared cards as the hard cap.
